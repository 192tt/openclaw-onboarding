# 角色专属采集项

## 1. OPC / 一人公司 / 独立创业者

| 字段 | 说明 |
|------|------|
| 核心 Skill / 主营业务 | 1～3 项，直接对接智能体 |
| 项目阶段 | 想法 / 原型 / 上线 / 盈利 / 规模化 |
| 当前融资需求 | 无 / 天使 / 额度范围 |
| 急需资源 | 技术 / 资金 / 渠道 / 供应链 / 法务财税 |
| 可对外输出能力 | 你能提供什么 Skill |
| 项目亮点关键词 | OpenClaw 自动提取 |

## 2. 投资人 / 天使机构

| 字段 | 说明 |
|------|------|
| 投资阶段 | 天使 / 早期 / 成长期 / 全阶段 |
| 主攻赛道 | 具体赛道名称 |
| 单笔可投金额 / 基金规模 | 金额范围 |
| 偏好项目类型 | AI / 技术 / 消费 / 产业等 |
| 可提供投后资源 | 导师 / 渠道 / 政策 / 对接 |
| 近期投资状态 | 活跃 / 观望 |

## 3. 孵化器 / 加速器 / 众创空间主理人

| 字段 | 说明 |
|------|------|
| 空间类型 | 综合孵化器 / 垂直赛道孵化器 / 产业园区 |
| 聚焦赛道 | 具体赛道名称 |
| 可提供政策 | 免租 / 补贴 / 投融资对接 |
| 配套服务 | 办公空间 / 导师 / 法务财税 |
| 所在具体区位 | 详细地址或区域 |
| 招募项目要求 | 阶段、赛道等 |

## 4. 企业需求方

| 字段 | 说明 |
|------|------|
| 企业所属行业 / 主营业务 | 行业描述 |
| 核心需求类型 | 技术采购 / 服务外包 / 项目合作 / 产品定制 / 生态联动 |
| 需求预算区间 | 金额范围 |
| 合作规模 | 短期项目 / 长期战略合作 / 年度采购 |
| 需求应用场景 | 内部升级 / 对外业务 / 市场拓展 |
| 对接负责人身份 | 企业负责人 / 采购总监 / 商务合作负责人 |
