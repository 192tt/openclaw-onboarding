# 数字身份卡片模板（6行精简版）

每张卡片固定显示 6 行核心信息：

```
┌─────────────────────────────────────────┐
│ [头像] 昵称                    [身份标签] │
│ 📍 城市  |  🏷️ 赛道1 · 赛道2 · 赛道3      │
│ ─────────────────────────────────────── │
│ 💬 一句话价值介绍（Slogan）               │
│ ─────────────────────────────────────── │
│ ⚡ 核心 Skill / 能力 / 投资方向           │
│ ─────────────────────────────────────── │
│ 🔍 我需要：xxx  |  💎 我能提供：xxx       │
│ ─────────────────────────────────────── │
│ 🤝 合作意向：[类型1] [类型2]  [匹配按钮]   │
│                                         │
│ [领域标签] [Skill标签] [需求标签] [匹配度] │
└─────────────────────────────────────────┘
```

## 按角色的第4、5行差异

### 创业者
- 第4行：`⚡ 核心 Skill：` + 主营业务（1~3项）+ `| 项目阶段：` + 阶段
- 第5行：`🔍 急需资源：` + 资源类型 + `| 💎 可输出：` + 能力

### 投资人
- 第4行：`⚡ 投资方向：` + 主攻赛道 + `| 阶段：` + 投资阶段
- 第5行：`🔍 寻找项目：` + 偏好类型 + `| 💎 投后资源：` + 可提供资源

### 孵化器
- 第4行：`⚡ 空间类型：` + 类型 + `| 聚焦：` + 赛道
- 第5行：`🔍 招募要求：` + 要求 + `| 💎 提供政策：` + 政策

### 企业需求方
- 第4行：`⚡ 所属行业：` + 行业 + `| 需求类型：` + 类型
- 第5行：`🔍 预算区间：` + 金额 + `| 💎 合作规模：` + 规模

## 智能标签栏（第7行）

根据采集内容自动生成：
- **领域标签**：基于赛道/行业提取（如 #AI #SaaS #消费）
- **Skill 标签**：基于核心能力提取（如 #技术开发 #投资 #孵化）
- **需求标签**：基于"我需要"提取（如 #找资金 #找技术 #找项目）
- **合作匹配度标签**：基于角色互补性预计算（如 #高匹配 #资源互补）
